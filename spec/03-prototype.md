# 页面原型

## 全局布局

```
┌────────────────────────────────────────────────┐
│ 🤖 AI 文本处理   首页 翻译 总结 历史      ☀️/🌙 │  ← 顶栏(导航 + 主题切换)
├────────────────────────────────────────────────┤
│                                                │
│              <RouterView 内容区>                │  ← 居中,max-width 960px,响应式
│                                                │
└────────────────────────────────────────────────┘
```

## 列表页 `/`

```
功能列表
选择一个功能开始使用

┌──────────┐ ┌──────────┐ ┌──────────┐
│ 中译英    │ │ 英译中    │ │ 文本总结  │   ← 卡片,点击进入;窄屏单列
│ 将中文…   │ │ 将英文…   │ │ 将长文本… │
└──────────┘ └──────────┘ └──────────┘
```

## 翻译页 `/translate`

```
文本翻译
[英译中 ▾]                          ← 语向选择
┌────────────────────────────────┐
│ 输入要翻译的文本…                │  ← textarea
└────────────────────────────────┘
[翻译] [停止生成] (running) 耗时 1234 ms
结果
┌────────────────────────────────┐
│ 你好…(打字机逐字)              │  ← StreamOutput(虚拟滚动)
└────────────────────────────────┘
```

## 总结页 `/summarize`

```
文本总结
要点数 [3 ▾]
┌────────────────────────────────┐
│ 粘贴需要总结的长文本…            │  ← 大 textarea
└────────────────────────────────┘
[总结] [停止生成] (running) 耗时 …
结果
┌────────────────────────────────┐
│ - 要点一                        │
│ - 要点二                        │
└────────────────────────────────┘
```

## 历史页 `/history`

```
调用历史                            [刷新]
记录每次调用的输入、输出、耗时与状态(数据闭环)
┌──────┬────────┬────────┬──────┬───────┬──────────┐
│ 功能 │ 输入    │ 输出    │ 状态 │ 耗时  │ 时间      │
├──────┼────────┼────────┼──────┼───────┼──────────┤
│ 翻译 │ Hello  │ 你好    │ done │ 1234  │ 06-15 …  │
└──────┴────────┴────────┴──────┴───────┴──────────┘
                         分页 10 条/页
```
